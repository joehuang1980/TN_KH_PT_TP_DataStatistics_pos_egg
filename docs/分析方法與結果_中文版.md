# 台灣四城市誘卵器陽性率與卵數之環境與人口因子分析

## 摘要

本研究針對台灣四個城市（台北、高雄、屏東、台南）進行誘卵器監測資料分析，探討環境與人口因素對蚊蟲活動的影響。透過統計相關性分析、主成分分析、聚類分析及時間滯後效應分析，識別出影響誘卵器陽性率（POS）及平均卵數（EGG）的關鍵因子，並比較跨城市差異。研究結果顯示溫度為最強預測因子，且不同城市呈現不同的環境敏感性模式。

**關鍵詞**：誘卵器監測、病媒蚊、環境因子、時間滯後分析、主成分分析

---

## 1. 前言

病媒蚊傳播的疾病對公共衛生構成重大威脅，特別是在亞熱帶及熱帶地區。誘卵器監測是評估蚊蟲密度的重要工具，能及早偵測蚊媒傳染病的潛在風險。環境因素如溫度、降雨量、植被指數，以及人口特徵如人口密度，皆可能影響蚊蟲的繁殖與活動。本研究旨在透過多城市比較分析，識別影響蚊蟲活動的通用預測因子及城市特異性因子，為病媒防治策略提供科學依據。

---

## 2. 材料與方法

### 2.1 研究設計與數據收集

本研究為多城市監測研究，分析台灣四個城市（台北、高雄、屏東、台南）的誘卵器監測資料。資料包含175個環境與人口特徵，以及兩個結果變量：誘卵器陽性率（POS）及平均卵數（EGG）。樣本數分別為台北3,264筆、高雄11,535筆、屏東2,799筆、台南82,805筆。

### 2.2 統計相關性分析

#### 2.2.1 方法選用

採用**Spearman秩相關係數（ρ）**分析175個特徵與兩個結果變量之間的關聯性。選用Spearman方法的原因如下：

1. **非參數特性**：不要求資料符合常態分布，適用於生態監測資料常見的偏態分布
2. **穩健性**：對異常值不敏感，減少極端值的影響
3. **單調關係檢測**：能捕捉非線性但單調的關係

#### 2.2.2 多重比較校正

使用**Bonferroni校正**控制家族錯誤率（FWER），顯著性水準設定為 α = 0.000286（0.05/175）。此嚴格校正確保在多重檢驗情境下的統計可靠性。

#### 2.2.3 特徵分組

特徵依據生態意義分為四類：
1. **溫度變量**：高溫、平均溫、低溫（含時間滯後）
2. **降雨變量**：降雨量（含時間滯後）
3. **植被指數**：NDVI平均值、中位數、總和（含時間滯後）
4. **人口變量**：人口密度、扶養比等

### 2.3 降維與聚類分析

#### 2.3.1 主成分分析（PCA）

對標準化特徵矩陣（z-score標準化）進行PCA降維，保留≥90%累積變異量。選用PCA的原因：

1. **降維效率**：將175個高度相關的特徵轉換為少數互相獨立的主成分
2. **去除共線性**：解決特徵間多重共線性問題
3. **保留資訊**：在大幅降維的同時保留主要資訊

各城市主成分數量由碎石圖（scree plot）和解釋變異比例決定。

#### 2.3.2 K-means聚類分析

在PCA降維資料上進行K-means聚類，識別不同的環境-人口剖面。最優聚類數（k）由三種驗證指標共同決定：

1. **Silhouette分數**（範圍：-1至1，越高越好）：衡量聚類內聚性與分離性
2. **Calinski-Harabasz指數**（越高越好）：評估聚類間分離度
3. **Davies-Bouldin指數**（越低越好）：評估聚類緊密度與分離度

測試k=2至10，每個k值進行100次隨機初始化以確保穩定性。

#### 2.3.3 聚類與結果變量關聯

使用**Kruskal-Wallis H檢驗**評估不同聚類間POS和EGG的差異。選用此非參數方法的原因：

1. **不要求常態性**：適用於非常態分布資料
2. **多組比較**：適合比較兩組以上的差異

當檢測到顯著差異（p < 0.05）時，進行**Dunn事後檢驗**配合Bonferroni校正進行兩兩比較。

#### 2.3.4 t-SNE視覺化

使用**t-distributed Stochastic Neighbor Embedding（t-SNE）**將高維聚類結果視覺化為二維空間（perplexity=30，1000次迭代），有助於理解聚類結構的複雜性。

### 2.4 時間滯後效應分析

#### 2.4.1 方法設計

分析環境變量與蚊蟲活動的時間關聯性，檢視1-12週滯後區間的相關性模式。兩種時間特徵類型：

1. **簡單滯後特徵**：t-n週前的值
2. **滾動窗口特徵**：t-n至t週的移動平均

#### 2.4.2 最優滯後期識別

對每個環境變量（高/平均/低溫、降雨、NDVI），計算各週滯後期與結果變量的Spearman相關係數，找出絕對相關係數最高的滯後週期。此分析針對每個城市獨立進行，以考量區域差異。

#### 2.4.3 視覺化

使用熱圖（特徵×滯後週數）和折線圖（相關強度×滯後期）呈現時間模式。顯著性以雙尾檢驗評估（p < 0.05）。

### 2.5 跨城市比較分析

#### 2.5.1 比較策略

跨4個城市進行統計比較，識別通用預測因子與城市特異性因子。結果變量分布使用Kruskal-Wallis H檢驗比較，後續進行Dunn事後檢驗配合Bonferroni校正。

#### 2.5.2 通用預測因子定義

特徵在所有城市皆須同時符合兩項標準：
1. |ρ| > 0.3（中度至強相關）
2. p < 0.001（極顯著）

#### 2.5.3 城市特異性因子定義

特徵僅在一或兩個城市符合上述標準。

#### 2.5.4 模式比較

比較聚類模式（最優k值、聚類-結果變量關係）和時間滯後效應（最優滯後期）以識別一致性與地點特異性模式。

## 4. 討論

### 4.1 溫度為主要預測因子

在所有城市中，溫度變量（特別是1-4週滾動平均溫）皆為最強或次強預測因子。這與蚊蟲生理學一致：溫度直接影響蚊蟲的代謝速率、發育時間、繁殖率及叮咬頻率。短期滯後效應（1週）反映溫度對成蚊活動的即時影響，而滾動平均則捕捉到累積溫度對族群動態的影響。

台北溫度相關性最強（ρ = 0.705），可能因其位於台灣北部，季節溫差較大，溫度變化對蚊蟲活動的限制作用更明顯。相對地，南部城市（高雄、屏東、台南）全年溫暖，溫度的限制效應較弱。

### 4.2 城市特異性模式

**高雄**呈現獨特模式，降雨成為最強預測因子，且整體相關強度較低。可能原因包括：
1. 都市化程度高，人為孳生源（如儲水容器）比自然環境更重要
2. 複雜的城市微氣候導致環境因子與蚊蟲活動關係較弱
3. 需要4個聚類反映更高的環境異質性

**屏東**顯示最短的時間滯後期（1.5週）和較高的NDVI負相關。屏東為農業縣，植被覆蓋可能影響微氣候與捕食者族群，進而影響蚊蟲密度。

**台南**樣本數最大（82,805），統計檢定力強，所有特徵皆達顯著。溫度與降雨皆重要，最優滯後期較長（6.2週滾動窗口），可能反映其複雜的農業-都市混合環境。

**台北**僅需2個聚類，環境剖面相對同質，但溫度相關性最強，顯示季節性對蚊蟲活動的強烈調控。

### 4.3 人口因素的負相關

所有城市中，人口密度、扶養比等人口因素皆與POS/EGG呈負相關。可能解釋：
1. **稀釋效應**：高人口密度區雖孳生源多，但單位面積誘卵器捕獲率可能因蚊蟲分散而降低
2. **環境管理**：高密度區公共衛生投入較多，病媒控制較積極
3. **都市化效應**：高度都市化區域的混凝土化降低自然孳生源

然而，此結果需審慎解讀，因誘卵器監測本身可能存在空間偏誤（如優先設置於高風險區）。

### 4.4 降雨的累積效應

降雨的最優滯後期較長（7-10週滾動窗口），顯示降雨影響蚊蟲密度需要時間累積：
1. 降雨形成孳生源需數日至數週
2. 蚊蟲從卵至成蟲需1-2週（依溫度而定）
3. 成蚊產卵至下一代成蟲又需1-2週

滾動窗口特徵優於簡單滯後，顯示累積降雨量（而非單週降雨）更能預測蚊蟲密度。

### 4.5 無通用預測因子的意涵

儘管溫度在所有城市皆重要，但未達到「通用預測因子」的嚴格標準（|ρ| > 0.3於所有城市），反映：
1. **地理異質性**：不同緯度、海拔、都市化程度導致環境因子影響差異
2. **防治策略差異**：各城市病媒防治投入與策略不同
3. **監測偏誤**：誘卵器設置密度與位置選擇可能因城市而異

這意味病媒防治策略應因地制宜，根據各城市特定環境特徵調整。

### 4.6 方法學優勢與限制

**優勢：**
1. **穩健的統計方法**：使用非參數方法（Spearman、Kruskal-Wallis）適應生態資料的非常態性
2. **嚴格的多重比較校正**：Bonferroni校正確保統計可靠性
3. **多角度分析**：相關性、聚類、時間滯後、跨城市比較提供全面視角
4. **大樣本驗證**：台南82,805筆資料提供高統計檢定力

**限制：**
1. **相關非因果**：相關分析無法建立因果關係，需實驗驗證
2. **聚類解釋性**：PCA轉換後的主成分難以直觀解釋
3. **時間解析度**：週度資料可能遺漏日尺度的快速變化
4. **空間異質性**：城市內部空間變異未充分探討
5. **蚊種未區分**：不同蚊種（埃及斑蚊、白線斑蚊）可能有不同環境偏好

## 5. 結論

本研究透過多城市、多方法的綜合分析，得出以下主要結論：

### 5.1 主要發現

1. **溫度為最一致的預測因子**：在所有城市中，溫度（特別是1-4週滾動平均溫）對誘卵器陽性率與卵數皆有強至中度相關（台北最強：ρ = 0.705），且呈現短期滯後效應（1週），顯示溫度對蚊蟲活動有立即且持續的影響。

2. **城市特異性顯著**：未發現所有城市共通的通用預測因子（|ρ| > 0.3），反映各城市在地理、氣候、都市化程度及防治策略上的差異。高雄以降雨為主要預測因子，台北以溫度為主，屏東呈現NDVI負相關，台南則溫度與降雨皆重要。

3. **降雨呈現累積效應**：降雨的最優滯後期為7-10週（滾動窗口），顯著長於溫度（1週），反映降雨通過孳生源形成、蚊蟲發育等多階段過程影響蚊蟲密度，且累積降雨量比單次降雨更具預測力。

4. **環境剖面與蚊蟲活動關聯**：PCA-聚類分析成功識別出不同環境-人口剖面（k=2-4），所有城市皆顯示聚類與POS/EGG有顯著關聯（p < 0.001），證實環境與人口因素組合對蚊蟲活動的重要性。

5. **人口因素呈負相關**：人口密度、扶養比等人口變量在多數城市與POS/EGG呈顯著負相關，可能反映都市化帶來的環境管理效應或稀釋效應。

6. **時間滯後期的地區差異**：屏東平均最優滯後期最短（1.5-3.5週），台南最長（6.2週滾動窗口），可能反映不同地理環境、蚊種組成與生命週期差異。

### 5.2 公共衛生應用

1. **即時預警系統**：溫度資料可作為短期（1週）蚊媒活動預警指標，特別在台北等溫度敏感城市
2. **季節性防治策略**：根據滾動平均溫度趨勢（1-4週）調整病媒防治強度
3. **降雨後防治窗口**：降雨後7-10週為防治關鍵期，應加強孳生源清除與化學防治
4. **因地制宜策略**：
   - 台北：重點監測溫度變化，季節交替期加強防治
   - 高雄：關注降雨模式，雨季後中長期監測
   - 屏東：考量植被管理與短期氣候變化
   - 台南：整合溫度與降雨預測，採用長期累積指標
5. **高風險區域識別**：利用聚類分析結果識別高風險環境剖面，優先配置防治資源

### 5.3 研究貢獻

1. **方法學創新**：整合統計相關、降維聚類、時間滯後及跨城市比較，提供病媒監測資料分析的系統性框架
2. **實證證據**：基於100,403筆誘卵器監測資料（四城市合計），提供台灣病媒蚊環境影響因子的實證基礎
3. **地區差異量化**：首次系統性比較台灣不同城市蚊蟲活動的環境驅動因子，為精準防治提供科學依據

### 5.4 未來研究方向

1. **因果推論研究**：透過實驗或準實驗設計驗證相關性背後的因果機制
2. **空間分析**：納入地理資訊系統（GIS）分析城市內部微尺度空間變異
3. **蚊種區分**：針對埃及斑蚊與白線斑蚊分別建立預測模型
4. **機器學習應用**：使用隨機森林、梯度提升等演算法提升預測準確度
5. **整合疾病資料**：結合登革熱等蚊媒傳染病發生率，建立環境-病媒-疾病完整鏈結
6. **氣候變遷情境**：模擬未來氣候情境下蚊蟲活動趨勢，為長期防治規劃提供依據
7. **成本效益分析**：評估不同環境監測與防治策略的成本效益比
8. **即時預測系統**：開發整合氣象資料與蚊媒監測的即時預警平台

### 5.5 政策建議

1. **建立跨城市監測網絡**：標準化監測方法與資料格式，促進跨區域比較
2. **整合氣象資料**：將即時與預報氣象資料納入病媒監測系統
3. **發展預警指標**：基於溫度與降雨累積建立量化預警閾值
4. **社區參與**：在高風險聚類區域加強社區教育與孳生源清除動員
5. **評估防治效果**：長期追蹤防治措施對環境因子-蚊媒關係的影響
